## ✔ library

1. fullpage : page animation library

   ```
   npm i @ap.cx/react-fullpage
   ```

2. react-slick : component animation library

   ```
   npm i react-slick
   ```

3. animate.css : element animation library

   ```
   npm i animate-css
   ```
